# Finance Dashboard

A simple finance dashboard I built using React. It helps track income and expenses, view transactions, and understand spending patterns.

## Features

* View total balance, income, and expenses
* Charts for balance trend and spending breakdown
* Add and delete transactions (admin only)
* Search and filter transactions
* Basic insights based on data
* Role-based UI (Viewer / Admin)
* Dark mode
* Responsive design
* Download report as PDF

## Tech Stack

* React
* Tailwind CSS
* React Router
* Recharts
* jsPDF
* Vite

## How to Run

```bash
npm install
npm run dev
```

## Approach

I kept the project simple and focused on core functionality.
Used React state (`useState`) instead of adding complex tools like Redux.

I also added a few extra features like dark mode, report generation, and a small intelligence mode to improve the experience.

## Notes

* Data is stored in localStorage
* Works on both desktop and mobile

That's it 🙂


**Built with a focus on simplicity (KISS principle)**
